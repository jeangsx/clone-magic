# Tema Shopify (GitHub Connect)

Esta carpeta es un **tema Shopify completo** (Shrine Pro + catálogo de ofertas).

## Conectar con GitHub

Shopify exige `layout/`, `config/`, `templates/` y `sections/` en la **raíz de la rama**.

No uses la rama `main` de `clone-magic` (es la app React).

Usa la rama **`theme`** de este mismo repo (contenido del tema en la raíz), o un repo aparte solo con estos archivos.

## Ofertas
- `sections/offers-catalog.liquid`
- `templates/page.offers.json`

En Admin: Páginas → Agregar página → plantilla **offers**.
